# Day 1: Random Variables & Expected Value

## Concept

A market return is an uncertain future number, so it can be modeled as a random variable. Expected value is the average outcome over many observations, not a guarantee about the next day.

## Market Connection

Daily Hang Seng Index returns can be treated as random variables, allowing us to estimate the average payoff and the distribution of possible outcomes.

## Mini-Project

Build an HSI Return Expectations Calculator.

## Outputs

- Expected daily return
- Expected annualized return
- Probability of positive and negative days
- Histogram of daily returns with expected value marked
